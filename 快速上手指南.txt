📌 这是什么东西？
─────────────────
一个基于 DeepSeek AI 的 QQ 聊天机器人，接入 QQ 群聊和私聊，
被 @ 就会回复，可以联网搜索、读取网页内容，性格是可爱的小猫娘。

内置可视化 Web 管理面板，浏览器里就能一键管理 Bot。

📥 第一步：下载安装所需软件
─────────────────────────────

你需要提前装好这些东西（都是免费的）：

  ① QQNT（QQ 9.9.x）
     → https://im.qq.com 下载安装
     ※ 注意：不要用 QQ 经典版，必须用 QQNT

  ② Python 3.10+
     → https://www.python.org/downloads 下载
     ※ 安装时记得勾选「Add Python to PATH」
     ※ 推荐 3.11 版本

  ③ NapCat Shell 版（QQ 机器人框架）
     → 加入 NapCat 官方群获取：
       https://github.com/NapNeko/NapCatQQ
     ※ 下载 Shell 版，不要下载 Console 版

  ④ DeepSeek API Key（AI 的 API 密钥）
     → https://platform.deepseek.com 注册账号
     → 在 API Keys 页面创建一个 Key
     ※ 免费的，注册就送额度

⚙️ 第二步：安装配置
─────────────────────

打开 PowerShell 或 CMD，依次执行以下命令：

  1. 克隆仓库
     git clone https://github.com/nico10086/-bot.git
     cd -bot

  2. 创建 Python 虚拟环境并安装依赖
     python -m venv env
     env\Scripts\pip install -r requirements.txt

  3. 配置 NapCat
     把下载的 NapCat Shell 版解压
     将解压出来的所有文件放到 -bot\NapCat.Shell\ 目录下
     （如果提示是否覆盖，选「是」）

  4. 配置 API Key
     在 -bot 文件夹里创建 .env 文件（用记事本就行）
     填入以下内容：

       DEEPSEEK_API_KEY=sk-你的密钥
       BOT_NAME=猫娘

     ※ 把「你的密钥」换成你刚才在 DeepSeek 官网申请的 Key

🚀 第三步：启动猫娘
─────────────────────

有两种方式，任选一种：

【方式一】双击 start_bot.bat（推荐）
  它会自动帮你完成以下流程：
    ✔ 找到你电脑上的 QQ
    ✔ 启动 NapCat 注入 QQ
    ✔ 弹出二维码 → 用手机 QQ 扫码登录
    ✔ 猫娘上线！
  看到终端出现 [WS] 已连接！等待 QQ 消息... 就说明成功了！

【方式二】使用 Web 管理面板（可视化）
  ① 双击 start_dashboard.bat
  ② 浏览器打开 http://127.0.0.1:5000
  ③ 点击「启动」按钮即可运行 Bot
  还可以在面板里查看日志、修改温度/模型等配置

❓ 常见问题
─────────────────

Q：报错「文件损坏，请重新安装QQ」？
  → 确认你装的是 QQNT（9.9.x），不是 QQ 经典版
  → 检查杀毒软件是否拦截了 NapCat

Q：双击 bat 后窗口一闪就没了？
  → 可能是没装 Python，或者没创建虚拟环境
  → 确认 env\Scripts\python.exe 存在

Q：二维码弹不出来？
  → start_bot.bat 会自动检测并弹出二维码
  → 如果没弹，手动打开 NapCat.Shell\cache\qrcode.png

Q：扫码后没反应？
  → 手机 QQ 上可能需要点「确认登录」
  → 检查端口 8080 是否被其他程序占用

Q：猫娘不回复群消息？
  → 检查是否在群里 @了猫娘
  → 检查 .env 里 BOT_NAME 是否配置正确

Q：猫娘回复太死板/太笨？
  → 当前默认 temperature=0.7，已经很灵活了
  → 如果还想更强，可以把模型改成 deepseek-v4-pro

💡 进阶小贴士
─────────────────

• 想让猫娘 24 小时在线？
  把电脑设置为不睡眠（电源设置），或者租一台云服务器

• 不想一直开着终端？
  用管理面板启动 Bot 后，可以关掉终端窗口吗？
  不行，但可以用 nssm 把 Bot 注册为 Windows 后台服务

• 猫娘崩了怎么办？
  start_bot.bat 有自动重启功能，挂了会自动重连

• 如何更新代码？
  在 -bot 目录下运行 git pull 即可

• 如何修改 Bot 的模型？
  ① 在管理面板的「快速配置」中修改模型名
  ② 或直接编辑 .env 文件，添加一行：
     MODEL=deepseek-v4-pro
  ③ 重启 Bot 生效
